# 📚 Smart Vocabulary Trainer 智慧單字訓練系統

> **課程名稱**：視窗程式設計 (II)
> **專案主題**：結合資料檔管理、多媒體播放與互動式操作之英語單字學習系統

---

## 📖 專案簡介

本專案以 C# Windows Forms 為開發平台，設計一套智慧化英語單字訓練系統。系統透過讀取 TSV 格式資料檔建立單字資料庫，並整合語音播放、自動學習模式與資料編輯功能，協助使用者建立更有效率的單字記憶流程。

在軟體架構上採用物件導向設計（Object-Oriented Programming），將單字資訊封裝為獨立類別，提升程式的維護性與擴充能力，同時實現資料載入、編輯與儲存等完整功能。

---

## ✨ 系統特色

### 🔊 真人語音學習

系統整合 Windows Media Player 元件，可根據單字資訊自動載入對應的 MP3 語音檔。

功能包含：

* 單字點擊即時發音
* 重複播放功能
* 自動播放模式
* 音訊資源動態載入

透過聽覺刺激搭配文字記憶，有助於提升單字學習效率。

---

### ⚡ 快速操作機制

為減少滑鼠操作頻率，系統加入鍵盤快捷控制：

| 快捷鍵          | 功能       |
| ------------ | -------- |
| Enter        | 顯示下一筆單字  |
| Space        | 重播目前單字發音 |
| Double Click | 編輯單字資料   |

透過 KeyPreview 技術實現全域鍵盤事件攔截，使操作更加流暢直覺。

---

### 🔄 自動學習模式

利用 Timer 控制項建立自動輪播功能。

系統可：

* 每隔固定時間切換單字
* 自動播放語音
* 自動移動至下一筆資料
* 保持目前項目於清單中央位置

讓使用者可在不操作電腦的情況下進行循環式記憶訓練。

---

### ✏️ 單字資料管理

使用者可直接於系統內修改單字內容，包括：

* 英文單字
* KK 音標
* 中文解釋
* 音檔路徑

修改完成後立即同步更新畫面，提升資料維護便利性。

---

### 💾 檔案儲存機制

系統提供完整資料回存功能。

透過 StreamWriter 將目前記憶體中的資料重新輸出為 TSV 格式檔案：

Word\tKK\tPath\tMeaning

使用者修改後無需額外處理，即可永久保存更新內容。

---

## 🏗️ 程式架構

### WordItem.cs

單字資料模型類別。

主要功能：

* 儲存單字資訊
* 解析 TSV 格式資料
* 轉換為文字輸出格式

---

### WordCollection.cs

單字集合管理類別。

主要負責：

* 檔案讀取
* 資料載入
* 集合管理
* 檔案寫入

---

### frmWordCards.cs

主系統介面。

功能包含：

* ListView 管理
* 語音播放
* 自動輪播
* 快捷鍵控制

---

### frmEditWord.cs

單字編輯視窗。

功能包含：

* 顯示原始資料
* 修改單字內容
* 回傳更新結果

## ⚠️ 使用說明

1. 將 WordCards.txt 放置於執行目錄。
2. 將 Sound 資料夾放置於專案根目錄。
3. 設定資源檔案屬性為「有更新時才複製」。
4. 執行程式後即可開始單字學習。

本系統適合作為英語單字記憶輔助工具，並展示 Windows Forms 應用程式在資料管理與多媒體整合上的實作能力。
